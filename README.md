This repository contains a minimal Node.js app.

Run the built-in HTTP server (no dependencies):

```bash
npm install   # optional but safe
npm start
# or
node index.js
```

Then open http://localhost:3000 or:

```bash
curl http://localhost:3000
```

Optional Express version:

```bash
npm install express
node app-express.js
```
